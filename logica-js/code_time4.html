Introdução as listas e filas

Estudantes

Crie um programa onde o usuário possa cadastrar estudantes sem limites, e, em seguida,
Se o usuário digitar "PARE" o programa deve exibir a quantidade de estudantes cadastrados e a lista com cada um deles.

Planetas

Crie uma array de planetas que inclua "Terra", "Marte", "Plutão", "Vênus", "Júpiter", "Saturno"
em seguida, peça ao usuário para digitar o nome de um planeta.

Compras

Crie uma array de frutas, exiba-a ao usuário e peça ao mesmo que escreva o nome de uma das frutas.
Se a fruta for achada retire-a e exiba a mensagem "Fruta foi retirada da lista".
